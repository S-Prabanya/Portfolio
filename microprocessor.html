Microprocessors

I have worked with advanced microprocessor platforms including RuggedBoard A5D2x and i.MX6UL. These experiences have enabled me to design complex systems with Linux and IoT capabilities.

RuggedBoard A5D2x

I used RuggedBoard A5D2x in a Smart Agriculture System, where I leveraged its processing capabilities for real-time data processing and cloud communication.

  • Linux-based development environment
  • GPIO, UART, and I2C interfacing
  • MQTT and cloud integration
  • Sensor data acquisition and control

i.MX6UL

With i.MX6UL, I explored Fledge for Industrial IoT edge data management. I developed applications that collect, buffer, filter, and forward data to cloud platforms.

  • Fledge plugin configuration
  • South and North plugin setup
  • Data buffering and filtering
  • REST API usage
  • Real-time edge-to-cloud data flow

These projects strengthened my skills in embedded Linux, IoT gateways, and real-time sensor-cloud communication using industrial-grade microprocessors.
