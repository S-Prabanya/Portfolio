Programming Languages I Work With

Overview

I have strong proficiency in both C and Python — two languages that complement my work in embedded systems, IoT projects, and AI applications. Whether it’s low-level firmware or high-level automation, I use these languages to build scalable and efficient solutions.

C Programming

C is my foundation for embedded system development. I use it primarily for real-time firmware design on microcontrollers like STM32.

Topics I’m confident with:

  • Conditional statements (if, switch)
  • Loops and control flow
  • Arrays and multi-dimensional arrays
  • Strings and string manipulation
  • Structures and unions
  • Pointers and pointer arithmetic
I’ve applied these concepts in real-time embedded projects such as a Smart Agriculture System, where C was used to interface sensors (e.g., DHT22) and control actuators.

Python Programming

Python enables me to handle high-level logic, automation, and AI workflows. It’s also my go-to for data processing and cloud communication.

Python topics I use regularly:

  • Control flow and loops
  • Functions and modules
  • Data structures (lists, tuples, dictionaries)
  • File handling and string processing
  • Serial communication using PySerial
  • Basic AI/ML logic using libraries like NumPy and scikit-learn
I used Python in both:
  • Smart Agriculture System: for serial data logging and dashboard integration
  • AI-based Pose Detection Project: for processing input data and applying simple ML logic

Project Highlights

  • Smart Agriculture System:
    • C for low-level hardware control
    • Python for data visualization and sensor monitoring
  • AI-based Pose Detection:
    • Python with OpenCV for image analysis
    • Simple ML algorithms to classify human pose

Conclusion

My expertise in C and Python allows me to bridge low-level embedded programming with high-level intelligent automation. I'm continuously exploring how these two languages can work together to solve real-world problems efficiently.
