Latest Project

Smart Street Light Project

Using STM32F446RE, RuggedBoard A5D2X, 4-Channel Relay, LCD-I2C Module, LoRa Module, MQTT,developed a smart street light system capable of real-time environmental monitoring and automated irrigation control.The system intelligently manages water usage based on dynamic soil moisture and ambient temperature levels, optimizing crop health and conserving resources. Sensor data is transmitted to the cloud via ThingsBoard for remote visualization, logging, and alerting. The firmware was developed using the Zephyr RTOS, ensuring low-power operation, modular design, and real-time performance.This solution demonstrates efficient edge-to-cloud integration for sustainable agriculture practices.

AI based Human Pose Detection

Developed a real-time Human Pose Detection system using the MoveNet model to extract 17 key body landmarks with high accuracy and low latency. Built a custom pose classification model on top of the keypoint data to accurately differentiate between various human poses such as standing, sitting. The system was optimized for lightweight inference on edge devices, making it suitable for applications like fitness tracking, gesture-based control, and rehabilitation monitoring. Emphasis was placed on robust keypoint preprocessing, temporal smoothing, and real-time classification accuracy.
