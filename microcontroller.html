Microcontrollers

I have hands-on experience with the STM32F446RE microcontroller. My knowledge spans across a wide range of its peripherals such as:

  • GPIO (General Purpose Input/Output)
  • USART / UART
  • I2C & SPI
  • ADC (Analog to Digital Converter)
  • TIMERS (PWM, Input Capture, Output Compare)
  • DMA (Direct Memory Access)
  • NVIC and Interrupt handling
  • Low Power Modes

Zephyr RTOS Experience

I have developed embedded systems using Zephyr RTOS, including a Smart Agriculture System where I used STM32F446RE and programmed it using Zephyr. Key features I handled include:

  • Device Tree configuration
  • Zephyr GPIO, I2C, UART APIs
  • Multithreading with k_thread
  • Sensor integration using Zephyr drivers
  • MQTT-based cloud communication
  • Make and CMake build systems

This experience not only strengthened my low-level embedded knowledge but also helped me develop production-grade code in real-time operating environments.
